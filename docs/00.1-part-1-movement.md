# Part 1 — Movement

Specifically, player movement!
This is the thing a player will be _constantly_ interacting with. So we want to get it right. 

We're going to tune the **Kinetics** to make it more fun and interesting to move around and jump. 

Then we will add some **Feedback** to help sell those movements.
