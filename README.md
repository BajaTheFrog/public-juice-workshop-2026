# 🍋 Your Game is a Lemon — JUICE IT 🍋

A hands-on workshop for **[GodotCon Boston](https://conference.godotengine.org/US/2026/) 2026**.

We got a game. We gonna JUICE it. 

## Prerequisites

**Godot 4.5**.
This project was built on and targets 4.5. Newer versions _probably_ work, but, can't say for sure! You can download 4.5 stable if you need to [here](https://godotengine.org/download/archive/). 

It's helpful if you know Git but, we can make it work without it!
And we have a special helper guide for Git as well: [Appendix: Git Help & FAQ](docs/14-appendix-git-help.md)

## Get the project

Clone the repository: 

```bash
git clone https://github.com/BajaTheFrog/public-juice-workshop-2026.git
```

Or if that sounds scary, download a zip of the project here:
https://github.com/BajaTheFrog/public-juice-workshop-2026/archive/refs/heads/main.zip


Then...
1. Launch Godot (4.5+).
2. Go to **Import** and navigate to where you cloned/downloaded the project. Select the `project.godot` file. 
3. Open the project.
4. Profit. 

## Run it
`Run Project` with **Cmd+B** or **F5**

OR 

Open `game/main.tscn` and run the current scene with **Cmd+R** or **F6**.

> Heads up: **Cmd+R** runs the current scene in the *Godot editor*. Once the game is
> running, **Cmd+R** resets the scene instead — see Controls below.

## Controls

| Action | Key |
| --- | --- |
| Move Left / Right | **A** / **D** |
| Jump | **W** — (Press / Hold) |
| Aim | **Mouse / Trackpad** |
| Shoot | **Left Mouse Button** — (Press / Hold) |

**Additionally**
| Action | Key |
| --- | --- |
| Open/Close the Juice menu | **J** |
| Reset the scene | **Cmd + R** |

There is more about the **Juice menu** [here](docs/08-appendix-juice-menu.md), but TL;DR its a handy tool for experimenting with the effects as we complete them. 

## Repo notes
There are only a few files you really need to be familiar with and they are identified at each step. So don't worry about getting your head around everything in the repo. That said, if you want to explore, go for it! I can't tell you what to do.
